# Urbanmile Construction Website

A responsive marketing website for Urbanmile Construction Private Limited built with React, Tailwind CSS, HTML, and JavaScript.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations
- Sections include:
  - Hero section with company overview
  - About (Company, Vision, Mission, Values)
  - Current Projects showcase
  - Why Choose Us
  - Directors profiles
  - Contact form
  - Footer

## Technologies Used

- React.js
- Tailwind CSS (via CDN)
- HTML5
- JavaScript (ES6+)

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
